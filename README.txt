- icon-192.png

## Como publicar no GitHub Pages (grátis)
1. Crie um repositório no GitHub (ex.: `calculadora-atraso`).
2. Envie **todos** os arquivos deste ZIP para a **raiz** do repositório.
3. No GitHub: Settings → Pages → *Build and deployment* → *Source*: **Deploy from a branch**.
   - Branch: `main` (ou `master`), Pasta: **/root**.
4. Acesse a URL gerada, por exemplo:
   - https://seuusuario.github.io/calculadora-atraso/

## Adicionar à tela inicial
- **Android (Chrome/Edge)**: menu ⋮ → **Adicionar à tela inicial**.
- **iPhone (Safari)**: botão **Compartilhar** → **Adicionar à Tela de Início**.

Pronto: o app abre em tela cheia, funciona **offline** e usa o **ícone** nos atalhos.
